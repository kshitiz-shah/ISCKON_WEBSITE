# ISKCON Pune Landing Page Design Guidelines

## Design Approach: Reference-Based
Drawing inspiration from spiritual/religious organization websites with emphasis on serenity, devotion, and community. The design should evoke peace, spirituality, and accessibility while maintaining modern web standards.

## Core Design Elements

### Color Palette
**Primary Colors:**
- Deep saffron: 30 85% 55% (spiritual/sacred)
- Rich burgundy: 350 70% 35% (devotional depth)
- Pure white: 0 0% 100% (purity/peace)

**Background Colors:**
- Warm cream: 45 25% 96% (light sections)
- Soft gray: 220 10% 95% (subtle backgrounds)
- Deep navy: 220 40% 15% (dark mode/footer)

**Accent Colors:**
- Golden yellow: 45 90% 60% (highlights/CTAs)
- Soft lotus pink: 340 30% 85% (gentle accents)

### Typography
- **Primary**: Crimson Text (serif) for headings - elegant, readable
- **Secondary**: Inter (sans-serif) for body text and UI elements
- **Accent**: Playfair Display for special spiritual quotes/mantras

### Layout System
Consistent spacing using Tailwind units: 4, 8, 12, 16, 24 for harmonious rhythm.
- Sections: py-16, py-24 for generous breathing room
- Content containers: px-4, px-8 with max-w-6xl centering
- Component spacing: gap-8, gap-12 between elements

### Component Library
**Navigation:** Clean header with centered logo, horizontal menu, prominent donation CTA
**Hero:** Full-width with overlay text, multiple deity images, blurred-background outline buttons
**Cards:** Soft shadows, rounded corners (rounded-lg), warm backgrounds
**Statistics:** Large numbers with descriptive text, icon accompaniments
**CTAs:** Golden buttons with subtle elevation, outline variants for secondary actions
**Content Sections:** Alternating backgrounds (cream/white) for visual separation

## Images
**Large Hero Image:** Yes - temple deity images with devotional atmosphere
**Gallery Images:** Temple activities, festivals, prasadam, devotees in service
**Icon Images:** Simple spiritual symbols, lotus motifs, service icons
**Background Images:** Subtle textures, mandala patterns for section dividers

## Visual Treatment
**Gradients:** Subtle warm gradients (saffron to gold) for hero overlays and CTA backgrounds
**Shadows:** Soft, warm shadows (not stark grays) to maintain peaceful atmosphere
**Borders:** Minimal use, prefer subtle background changes for section separation
**Spacing:** Generous whitespace to create contemplative, uncluttered experience

## Content Strategy
- **Hero:** Immediate spiritual welcome with clear value proposition
- **5 Core Sections:** Teachings, Community Stats, Daily Schedule, Donation, Gallery
- **Single Focus:** Spiritual growth and community connection
- **Breathing Room:** Each section fully digestible before scrolling

The design should feel welcoming to newcomers while honoring traditional spiritual aesthetics.
